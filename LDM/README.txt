The package includes the MATLAB/C++ code of the learning algorithm LDM, which optimizes the margin distribution [1].

[1] T. Zhang and Z.-H. Zhou. Large margin distribution machine. In: Proceedings of the 20th ACM SIGKDD Conference on Knowledge Discovery and Data Mining (KDD'14), New York, NY, 2014, pp.313-322.


For LDM, you can call the "LDM" function to run the algorithm. The setting for the parameters can be found in the file "LDM.m".

You will find a demo of using this code in the root directory. The example data is "heart" and "real-sim".
